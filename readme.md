<p>
有一个专门保存pixiv投稿的数据库以及一系列配套的多线程调用方法<br/>
数据库管理目前正在逐步减少和背景变换模块的代码交叉/依赖<br/>
用学了不到一周的wpf xaml写动态窗体<br/>
能用API修改桌面壁纸，实现定时更新<br/>
<br/>
初次使用需要指定背景目录才能自动更换背景:D
